发表于 2020-11-06 分类于 python 阅读次数：
sublime插件

装机

1.convertToUTF8

功能说明：ConvertToUTF8 能将除UTF8编码之外的其他编码文件在 Sublime Text 中转换成UTF8编码，在打开文件的时候一开始会显示乱码，然后一刹那就自动显示出正常的字体，当然，在保存文件之后原文件的编码格式不会改变。

2.BracketHighlighter 插件

功能说明：高亮显示匹配的括号、引号和标签。

3.JsFormat 插件

功能说明：JavaScript代码格式化。

使用方法：在打开的JavaScript文件里点右键，选择JsFormat

4.Alignment 插件

功能说明：使代码格式的自动对齐。

安装方法：Ctrl+Shift+P面板，安装插件。

使用方法：快捷键 Ctrl+Alt+A

设置注释颜色

1.安装插件resource view

2.打开package resource viewer:open resource

3.查找color scheme-default

4.选择monokai，即可打开monokai.sublime-color-scheme

1.编译环境

1、打开Tools > Build System > New Build System..

2、点击New Build System后，会生成一个空配置文件，在这个配置文件内覆盖配置信息，本文python安装路径为“D:/Anaconda3”，（注意区分正反斜杠，请将路径换成python实际安装路径），然后按ctrl+s，将文件保存在默认路径，文件名命名为“Python3”
